λ 递归

(lambda (x) (x x))

这个例子相当引人入胜，用 λ 构造出 length 来计算 list 的长度：

(((lambda (mk-length)

              (length (cdr l))))))))

这个仅能计算一层，展开的话是：

->

  (lambda (l)
    (cond
      ((null? l) 0)
      (else (add1
            (length (cdr l)))))))

    (cond

    (else (add1
        ((lambda (length)

            (cond
              ((null? l) 0)
              (else (add1
                (length (cdr l)))))))
           (cdr l))))))                 ; 本应传递一个 `length` 得到内部的 lambda，但传递了一个 `list`
  '())

->
0

可以看出来尚不足以进行长度大于 0 的递归，因为到后面 length 没有正确的参数传进去。

修复

; 一步步改进
(((lambda (mk-length)
   (mk-length mk-length))
 (lambda (mk-length)
   (lambda (l)
     (cond
       ((null? l) 0)
       (else (add1
              ((mk-length eternity)
               (cdr l))))))))
 '(1))

这里放了个 eternity 虚位以待，然而并不能改变它算不了非空 list 的事实：
